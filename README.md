# Prerequisite
 * NodeJS
 * Node Package Manager (npm)
 * Package modules
 
```
npm install express
npm install node-async-loop
npm install config
```

# Lunching the Web App
 * Run the node server.js with npm start.
 
```
npm start

> node server.js

listening to http://127.0.0.1:8899
```


# Test the service
 * Browse http://localhost:8899/
